# Wi-Fi Troubleshooting Guide

## 🔍 Masalah Umum

* Tidak bisa konek ke Wi-Fi
* Internet tidak jalan
* Koneksi lambat

---

## 🛠️ Langkah Troubleshooting

### 1. Cek Koneksi Fisik

* Pastikan router menyala
* Cek kabel LAN terhubung dengan benar

### 2. Restart Perangkat

* Restart router
* Restart komputer / HP

### 3. Cek IP Address

* Gunakan command:
  ipconfig
* Pastikan mendapatkan IP (bukan 169.x.x.x)

### 4. Test Koneksi

* Ping ke Google:
  ping 8.8.8.8

### 5. Cek Provider Internet

* Pastikan tidak ada gangguan dari ISP

---

## ✅ Solusi Umum

* Restart router biasanya menyelesaikan 60% masalah
* Reset network adapter jika masih gagal

---

## 📌 Catatan

Selalu dokumentasikan masalah yang sering terjadi untuk mempercepat troubleshooting berikutnya.
